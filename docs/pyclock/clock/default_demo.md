# 出厂例程

在资料包示例程序里面有一个出厂例程，该例程整合了AP配网、WiFi连接、NTP时钟、天气信息获取以及天气时钟UI界面，是pyClock的出厂例程代码。以便用户在编程学习后可以恢复。这些功能实际上是将前面教程内容整合，用户可以自行研究相关代码。

![default_demo](./img/default_demo/default_demo1.png)

将所有代码和素材通过Thonny IDE发送到pyClock即可。
