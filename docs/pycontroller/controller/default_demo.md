# 出厂例程

在资料包示例程序里面有一个出厂例程，该例程整合了NES游戏、pyCar遥控车、pyDrone四轴飞行器和手柄控制测试功能，并配有图例UI，以便用户恢复出厂代码。这些功能实际上是将综合实验内容整合，用户可以自行研究相关代码。

![default_demo](./img/default_demo/default_demo1.png)

将所有代码和素材通过Thonny IDE发送到pyController即可。
