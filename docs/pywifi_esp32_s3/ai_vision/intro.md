# 简介

乐鑫ESP32-S3是乐鑫2016年推出ESP32以来第三代IC，在全球万物互联向万物智联快速升级的趋势下，人工智能 (Artificial Intelligence) 作为发挥技术潜能的主动力，在 AIoT 领域占据着越来越重要的地位。为响应市场对 AI 算力的技术需求，乐鑫推出了 ESP32-S3 芯片，精准聚焦 AIoT 市场。

ESP32-S3 MCU 额外增加了用于加速神经网络计算和信号处理等工作的向量指令 (vector instructions)。AI 开发者们将可以使用指令优化后的软件库，实现高性能的图像识别。

本章主要讲述AI的机器视觉应用，内容涵盖颜色识别、运动检测、猫脸检测、人脸检测、人脸识别等。01Studio已经将上述功能移植到MicroPython固件，用户使用pyWiFi-ESP32-S3开发板和固件可轻松实现上述功能，**需要注意的是需要使用8M内存版+AI固件**。
